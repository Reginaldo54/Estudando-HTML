Tags no HTML

O que seria uma tag ? É uma instrução que damos ao html, de como queremos que o contéudo dentro delas se pareça ou que tipo de função ela deve exibir dentro do site que estamos fazendo.

Elas são bastante usadas para principalmente estruturar e organizar o código html. Existem varios tipos de tags e com isso vários usos, vou citar então alguns tipos mais usados no html, mas antes disso é preciso entender como é a estrutura de uma tag. 

Anatómia: 
Abrindo a tag -> <p>;
Contéudo da tag -> é tudo aquilo que fica dentro, entre o ínicio e o fim da tag.
Fechando a tag -> </p>; 

A junção desses 3, é denominada -> elemento. Vou entrar em detalhes sobre isso após
falar um pouco mais sobre os tipos de tag.

Tipos de Tags

Tag de bloco

Um bloco é a junção de linhas (duas ou mais), Tags de bloco são tags que criam um espaço (margem) entre os elementos anteriores definidos e os que vão ser criados abaixo deles.

Exemplo de tags de bloco: p, h1 ao h6, div.

Tag de linha (inline)

São tags que não abre nenhum espaçamento entre elementos anteriores, tomando em conta que o html deixa todo o seu contéudo alinhado, ou seja, não importa o que eu escreva dentro do html, se eu não colocar uma tag que quebra linhas como br (break) ou uma tag de bloco, ele não adicionará nenhuma separação dos elementos, o contéudo do site iria ficar muito semelhante a uma redação, que não pula nenhuma linha.

Exemplo de tags de linha: a, b, strong, i, em, u.

Tags mais usadas:

Tag Função Tipo
p É uma tag que cria um paragráfo dentro do html. Bloco
a É uma tag que cria links para outros sites ou para uma parte da mesma página! O 'a' vem de anchor Linha
b ou strong Deixam o contéudo dentro delas em negrito Linha
i ou em São tags que deixam a fonte do contéudoo no formato itálico. Linha
abbr Abrevia em uma sigla, ao colocar o mouse em cima da sigla irá mostrar o significado da sigla Linha
h1 à h6 elas deixam o contéudo dentro delas em negrito e com um fonte maior que a fonte padrão! Bloco

Para mais tags segue o link.

Elementos dentro do HTML

Então, o que seria um elemento? Um elemento é dito ser uma tag do html.
E por isso existe um elemento chamado elemento vázio. É dado este nome devido a ele não ter um ínicio ou fim .
Exemplo De elementos vázios: Tag img, link, input.

Fluxo de elementos do HTML

Como bem sabemos todos os códigos em html tem uma sequencia, o titulo por exemplo, aparece antes do que seu contéudo. Logo os elementos criados primeiro executados/processados primeiro que os elementos abaixo deles. Tudo depende do posicionamento dos elementos.

Hierarquia dos elementos dentro do HTML

Dentro do html é possível criar uma tag dentro de outra, isto é chamado de aninhamento de tags. Nesse aninhamento as tags dentro de outra são chamadas de tags filhas, e consequentemente a outra é chamada de tag mãe/pai.

Esse aninhamento permite que durante a formatação dos elementos no CSS, o elemento seja facilmente localizado a apartir de seu "pai".


Tipos de atributos

Atributos globais

São atributos genéricos, ou seja, podem ser usados em todos as tags existentes. Exemplo: style, title, id, class, hidden.

Atributos Específicos

São atributos que apenas existem em determinadas tags, podendo até ser únicos. Exemplo: type (da tag input), alt (da tag img), href (da tag a).

Atributos boleano

Alguns atributos podem ser ativados ou desativados, eles são chamados de boleanos. Apesar de na maioria das vezes não serem atribuidos valor boleano, já que uma vez que ele é chamado, ele já está funcionando de acordo com sua função.
Ex: hidden, ele vai ocultar o elemento que chamou ele, mas pode-se colocar hidden = "false" (tornando o elemento visivel novamente...). Mas, ao não chamar o hidden, ele já é visivel, logo é desnecessário deixar ele como false.

Exemplo de tags boleanas: hidden, editablecontent
